# 喝水提醒小桌宠

Windows桌面常驻的喝水提醒小工具。一只透明悬浮的小水滴,平时安静呼吸,到点会跳起来叫你喝水。

## 是什么

- 形态:透明悬浮窗+系统托盘,不占任务栏
- 提醒:固定间隔(默认45分钟),可在设置里改 30/45/60/90
- 交互:鼠标悬停展开操作面板(已喝/再等10分钟),点击托盘图标进设置
- 灵魂:非线性动效、丝滑过渡、不打扰但有存在感

## 怎么跑

```bash
npm install
npm run dev
```

启动后会在屏幕右下角出现一只小水滴,系统托盘出现图标。右键托盘图标可以打开设置或退出。

## 怎么打包

```bash
npm run build
```

产物在 `dist/` 下,Windows安装包(.exe)。

## 配置

设置面板(右键托盘 → 设置)里能改:

- 提醒间隔:30 / 45 / 60 / 90 分钟
- 开机自启:开/关
- 提醒声音:开/关

配置存在 `%APPDATA%/water-reminder/config.json`。

## 技术栈

Electron + Vue3 + Vite + Tailwind + GSAP。详见 [CLAUDE.md](./CLAUDE.md)。

## 目录

- `electron/` 主进程(窗口/托盘/定时器)
- `src/` 渲染进程(Vue 组件)
- `docs/` AC 和设计方案
