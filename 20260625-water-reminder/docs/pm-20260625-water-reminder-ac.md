# MVP 功能 AC + 动效方案

## 范围声明

MVP 只做提醒能力本身。不做:数据记录、历史趋势、多种形象切换、自定义文案。

## 功能 AC

### 1. 桌宠常驻

- **AC-1.1** 应用启动后,屏幕右下角(距屏幕右边/下边各 40px)出现一个 120x120 的透明悬浮窗,内含水滴形态的小桌宠
- **AC-1.2** 悬浮窗不在任务栏显示,不抢焦点,始终置顶
- **AC-1.3** 鼠标可拖拽悬浮窗,松手后位置持久化到配置,下次启动恢复
- **AC-1.4** 系统托盘出现图标,右键菜单:打开设置 / 现在就喝一杯 / 退出

### 2. 定时提醒

- **AC-2.1** 默认每 45 分钟触发一次提醒
- **AC-2.2** 触发时桌宠播放"召唤序列"动效(见下方动效方案 §B),系统通知同时弹出(标题"该喝水了",正文"已经 XX 分钟没喝了")
- **AC-2.3** 配置改间隔后,下一轮立即按新间隔计时
- **AC-2.4** 用户点"已喝"或托盘"现在就喝一杯"后,计时器重置

### 3. 桌宠交互

- **AC-3.1** idle:桌宠 4 秒一个循环的呼吸动画(scale + opacity,sine 缓动)
- **AC-3.2** hover:鼠标进入悬浮窗时,桌宠放大到 1.15,出现轻微的跟随鼠标位移(最大偏移 8px)
- **AC-3.3** hover 持续 300ms 后,从桌宠侧边滑出操作面板(已喝 / 再等 10 分钟 / 设置 三个按钮)
- **AC-3.4** 鼠标离开后,面板 spring 回收,桌宠回到 idle
- **AC-3.5** 双击桌宠 = 已喝(快捷操作)

### 4. 设置面板

- **AC-4.1** 通过托盘"打开设置"或桌宠面板"设置"按钮唤起,独立窗口,不置顶,玻璃拟态
- **AC-4.2** 间隔选择:30/45/60/90 分钟,iOS 风格分段控件
- **AC-4.3** 开机自启:开关项,改后立即生效(写注册表)
- **AC-4.4** 提醒声音:开关项,关闭后系统通知静音
- **AC-4.5** 修改实时保存,不需要点"保存"按钮

### 5. 系统集成

- **AC-5.1** 单实例,重复启动只显示已运行的桌宠
- **AC-5.2** Windows 系统通知用原生 toast,不自绘
- **AC-5.3** 关闭主窗口不退出应用,只有托盘"退出"才真退出

## 动效方案

### A. 桌宠形态

SVG 绘制:

- 主体:水滴形 path,渐变填充 #4FB7E0 → #5FD9CB(自上而下)
- 高光:左上 30° 椭圆高光,白色半透明 30%
- 阴影:外侧 drop-shadow(0 8px 24px rgba(79, 183, 224, 0.4)),给悬浮感
- 眼睛(可选):两个深色小圆点,赋予生命感。MVP 阶段先不加眼睛,纯几何水滴。

### B. 召唤序列(提醒触发,GSAP timeline)

总时长 1000ms,四阶段:

```
t=0     scale(1) → scale(0.85)        缓动 power2.in     180ms  (预备压缩)
t=180   scale(0.85) → scale(1.3)      缓动 back.out(2)   220ms  (弹起,过冲)
t=400   scale(1.3) → scale(1.1)       缓动 elastic.out   300ms  (弹性回弹)
t=700   scale(1.1) → scale(1.05)      缓动 sine.inOut    300ms  (稳定到呼吸态)

并行: t=0   颜色 #4FB7E0 → #FFA94D     缓动 sine.inOut    400ms  (蓝→橙,提示色)
并行: t=600 颜色 #FFA94D → #4FB7E0     缓动 sine.inOut    400ms  (恢复)
并行: t=200 浮出文字"喝水啦"            缓动 back.out      400ms  (面板上方)
```

### C. 呼吸(idle)

GSAP repeat -1, yoyo:

```
scale: 1 → 1.04          缓动 sine.inOut    2000ms
opacity: 0.92 → 1        缓动 sine.inOut    2000ms
```

### D. 鼠标悬停跟随

mousemove 监听窗口内坐标,桌宠 transform 平滑跟随:

- 偏移量 = (鼠标坐标 - 窗口中心) * 0.1,最大 ±8px
- GSAP quickTo(x, y, { duration: 0.4, ease: 'power3.out' }),响应顺滑不卡顿

### E. 面板展开

```
hover 触发后 delay 300ms:
  面板 x: +60px → 0     缓动 back.out(1.4)  400ms
  面板 opacity: 0 → 1   缓动 power2.out     250ms
  按钮 stagger 入场:    每个 delay 50ms, 同样缓动
```

收回时全部反向,duration 缩短到 250ms(出场更快)。

### F. reduced-motion 降级

`window.matchMedia('(prefers-reduced-motion: reduce)')` 为 true 时:

- 所有 scale/位移动画 → fade only
- 召唤序列 → 仅颜色闪烁 + 通知
- 呼吸 → 静止
